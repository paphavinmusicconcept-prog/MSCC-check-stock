# วิธีติดตั้งบอทเช็คสต็อก

---

## ส่วนที่ 1 — คอมที่ร้าน (api.py)

### ขั้นตอนที่ 1: ติดตั้ง Python
1. เข้า https://www.python.org/downloads/
2. กด "Download Python" แล้วติดตั้ง
3. **สำคัญ**: ติ๊ก ✅ "Add Python to PATH" ก่อนกด Install

### ขั้นตอนที่ 2: ติดตั้ง library
เปิด Command Prompt (กด Win+R พิมพ์ cmd กด Enter) แล้วพิมพ์:
```
pip install flask mysql-connector-python
```

### ขั้นตอนที่ 3: แก้ไข api.py
เปิดไฟล์ `api.py` แก้ค่าเหล่านี้ให้ตรงกับของร้าน:
- `user` — ชื่อผู้ใช้ MySQL (ถามจากคนติดตั้ง Express)
- `password` — รหัสผ่าน MySQL
- `database` — ชื่อฐานข้อมูล
- `TABLE`, `COL_SKU`, `COL_NAME`, `COL_QTY` — ชื่อตารางและคอลัมน์

### ขั้นตอนที่ 4: รัน api.py
เปิด Command Prompt ในโฟลเดอร์ที่เก็บไฟล์ แล้วพิมพ์:
```
python api.py
```
ถ้าขึ้น "Local API พร้อมแล้ว" แสดงว่าสำเร็จ

### ขั้นตอนที่ 5: ติดตั้งและรัน ngrok
1. เข้า https://ngrok.com → สมัครฟรี → ดาวน์โหลด ngrok สำหรับ Windows
2. แตกไฟล์ได้ ngrok.exe
3. เปิด Command Prompt ใหม่อีกหน้าต่าง พิมพ์:
```
ngrok http 5000
```
4. จะได้ URL เช่น `https://abcd-1234.ngrok-free.app`
5. **คัดลอก URL นี้** ไปใส่ใน bot.py บรรทัด `LOCAL_API_URL`

## ส่วนที่ 2 — LINE Bot (bot.py)

### ขั้นตอนที่ 6: สร้าง LINE OA และ Messaging API
1. เข้า https://developers.line.biz
2. สร้าง Provider → สร้าง Channel (Messaging API)
3. คัดลอก **Channel Secret** และ **Channel Access Token**
4. ใส่ค่าทั้งสองใน bot.py

### ขั้นตอนที่ 7: Deploy bot.py
แนะนำ Render.com (ฟรี):
1. อัพโหลดโค้ดขึ้น GitHub
2. เข้า https://render.com → New Web Service → เลือก repo
3. Start Command ใส่: `python bot.py`
4. จะได้ URL เช่น `https://my-bot.onrender.com`

### ขั้นตอนที่ 8: ตั้ง Webhook URL
1. กลับไป LINE Developers
2. ใส่ Webhook URL: `https://my-bot.onrender.com/callback`
3. กด Verify → ควรขึ้น Success

## ปัญหาที่พบบ่อย
- **api.py ไม่ติดต่อ MySQL** → ตรวจสอบ user/password/database ใน api.py
- **บอทไม่ตอบ** → ตรวจว่า ngrok รันอยู่ และ URL ใน bot.py ถูกต้อง
- **ngrok URL เปลี่ยนทุกครั้ง** → ต้องอัพเดท LOCAL_API_URL ใน bot.py ใหม่ทุกครั้ง (แก้ได้ด้วย ngrok plan ที่มี static domain)
